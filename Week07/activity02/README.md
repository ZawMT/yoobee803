## Week 7 - Activity 2: Fraud Detection Project
Use the below dataset to perform a classification task for fraud detection. You may use the Support Vector Machine (SVM) model to classify and detect fraudulent records. Before applying the machine learning model, make sure you complete all required data preparation and data cleaning steps, including handling missing values, removing duplicates, feature selection, encoding, normalization/scaling, and exploratory data analysis (EDA).

Prepare your final results as a PowerPoint presentation that includes:
- Introduction to the dataset and problem statement
- Data preprocessing and cleaning steps
- Exploratory data analysis and visualizations
- ML model development and implementation
- Training and testing process
- Evaluation metrics such as Accuracy, Precision, Recall, F1-Score, Confusion Matrix
- Discussion of results and findings
- Conclusion and future improvements

Your presentation should be suitable for a 10-minute recorded-demo presentation of your outcome.
In addition, share your GitHub repository including:
- Source code
- Cleaned/staging dataset
- PowerPoint presentation
- Short report with data visualizations (in 2 pages)
- Recorded video demo explaining your project and results

See help link below for more information about the dataset: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
