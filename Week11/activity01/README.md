## Week 11 - Activity 1.1: predication project 
Develop a **prediction pipeline** for the dataset using **Linear Regression, XGBoost, LSTM, Artificial Neural Network (ANN), and ARIMA** models. Compare the performance of all models and present the findings in a **PowerPoint presentation**.
The project should include:
-  Data understanding and preprocessing
- Exploratory Data Analysis (EDA)
-  Visualization of input data distributions and key features
- Feature engineering and data preparation
- Model development using:
    - Linear Regression
    - XGBoost
    - Artificial Neural Network (ANN)
    - Long Short-Term Memory (LSTM)
    - ARIMA
- Model evaluation and comparison using appropriate performance metrics (e.g., RMSE, MAE, R², MAPE)
- Comparative results presented in tables and visualizations
- Discussion of model strengths, limitations, and performance differences
- Key insights and recommendations

**PowerPoint Presentation Requirements**:
- Brief introduction to the dataset and project objectives
- Data distribution and exploratory visualizations
- Methodology and prediction pipeline
- Model comparison results
- Visual representation of actual vs. predicted values
- Summary of key findings and insights
- Conclusions and recommendations
